## 2026-03-25

### 모바일 스와이프 탭 전환 기능 추가
- 홈 / 납입현황 / 투자현황 / 여행투표 탭을 좌우 스와이프로 전환 가능
- 수평 이동 50px 이상, 수직보다 수평이 클 때만 탭 전환되어 스크롤과 충돌 없음
- 별도 라이브러리 없이 순수 터치 이벤트로 구현

### 구글 시트 연동 (읽기 전용)
- 납입현황, 투자현황 데이터를 구글 워크시트에서 직접 읽어오도록 변경
- 기존 하드코딩 데이터 → Google Sheets CSV export API 방식으로 전환
- 납입현황 시트 (gid=0), 투자현황 시트 (gid=278279642)
- 앱에서는 읽기만 하고, 데이터 수정은 구글 시트에서만 진행
- localStorage 저장 제거 (시트가 원본 데이터)
- fetch 실패 시 초기 하드코딩 데이터로 폴백

### 모바일 당겨서 새로고침 (Pull-to-Refresh) 기능 추가
- 페이지 최상단에서 아래로 당기면 구글 시트 데이터 재로드
- 80px 이상 당기면 "↑ 놓으면 새로고침" 표시
- 새로고침 중 스피너 인디케이터 표시
- 좌우 스와이프(탭 전환)와 충돌 없이 동작
